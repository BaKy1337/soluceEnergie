Collecte des données personnelles

Les informations recueillies sur nos formulaires sont enregistrées dans un fichier informatisé. Nous collectons les données suivantes : nom, prénom, adresse e-mail, et numéro de téléphone.

Les données collectées seront uniquement communiquées à notre équipe chez Soluce Energie.

Nous conservons vos données pendant la durée légale. Cependant, sur demande, vous avez le droit de demander la suppression de vos données.

Vous pouvez demander l’accès aux données vous concernant, les rectifier, demander leur effacement, ou exercer votre droit à la limitation du traitement de vos données. Vous pouvez également vous opposer au traitement de vos données ou exercer votre droit à la portabilité de vos données.

Consultez le site cnil.fr pour plus d’informations sur vos droits.

Pour exercer ces droits ou pour toute question sur le traitement de vos données dans ce dispositif, vous pouvez nous contacter par e-mail à l'adresse contact@soluce-energie.fr.
